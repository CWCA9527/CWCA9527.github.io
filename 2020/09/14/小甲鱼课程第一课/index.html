课后习题

  1. Python 是什么类型的语言？
    解释型、面向对象、动态数据类型

  2. IDLE 是什么？
    集成开发环境，键入文本和程序的交互途径

  3. print() 的作用是什么？
    打印输出

  4. Python 中表示乘法的符号是什么？
    *

  5. *为什么 >>>print(‘I love fishc.com ‘ * 5) 可以正常执行，但 >>>print(‘I love fishc.com ‘ + 5) 却报错*？
    字符串无法和数字相加，python中不能把两个完全不同的东西相加

  6. 如果我需要在一个字符串中嵌入一个双引号，正确的做法是？
    利用反斜杠（\）对双引号转义："，或者用单引号引起这个字符串。例如：’ I l”o”ve fishc.com ‘。

  7. 为什么我们要使用 Python3？Python2到底有什么问题？看起来很多程序员依然都在使用Python2？
    python2和python3的不兼容性
