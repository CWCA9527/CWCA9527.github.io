第一天

random 模块

  • 描述

    random（）方法为随机返回一个实数，其范围在[0，1）内。

  • 语法

    import random

    random.random()

  • 参数

    无

  • 返回值

    在[0,1)内

  • 实例

    import random
    secret = random.random() 得到结果

    print (secret) 打印得到的结果
    以上实例得到结果：0.2964213966154332

  • random用法

    random各种使用方法

      import random

    随机生成[0.1)的浮点数

      print("random():", random.random())

    随机生成1000-9999之间的整数

      print("randint(1000, 9999):", random.randint(1000, 9999))

    随机生成0-20之间的偶数

      print("randrange(0, 21, 2):", random.randrange(0, 21, 2))

    随机生成0-20之间的浮点数

      print("uniform(0, 20):", random.uniform(0, 20))

    从序列中随机选择一个元素

      list_string = ['a', 'b', 'c', 'd', 'e']

  print("choice(list):", random.choice(list_string))

  print("choice(string):", random.choice('abcd'))

    对列表元素随机排序

      list_number = [1, 2, 3, 4, 5]

  random.shuffle(list_number)

  print("shuffle(list):", list_number)
