前言

随着自己兴趣的转变与拓展，开始逐渐爱上了Python语言的魅力，也想好好利用研一的空闲时间完成对于Python语言的学习与提高。梦所在的方向，必是自己所热爱的地方。

一、计算机与程序设计

1. 计算机的概念

概念：计算机是根据指令操作数据的设备

  • 功能性：对数据的操作表现为数据的计算、输入输出处理和结果储存等。

  • 可编程性：根据一系列指令自动的、可预测的、准确的完成操作者的意图。

    2. 计算机的发展

    发展：计算机的发展遵循摩尔定律，表现为指数方式。

  • 计算机硬件所依赖的集成电路规模参照摩尔定律发展

  • 摩尔定律：单位面积集成电路上可容纳的晶体管数量约每两年翻一番

  • CPU/GPU、内存、硬盘都遵循摩尔定律

    3. 程序设计

程序设计是计算机可编程性的体现

编程语言用于人与计算机之间的交互

4. 编译和解释

计算机执行程序的两种方式：编译和解释
编译：将源代码一次性转换成目标代码的过程，执行编译过程的程序叫做编译器。

  • 一次性翻译，之后不再需要源代码（类似于英文翻译）

解释：将源代码追条转换成目标代码同时逐条运行的过程，执行解释过程的程序叫解释器。

  • 每次程序运行时随翻译随执行（类似于同声传译）

源代码：采用某种语言编写的计算机程序，人类可读。
目标代码：计算机可执行，人类不可读（二进制数）。

5. 静态语言和脚本语言

静态语言：使用编译执行的编程语言。

  • C/C++、Java

脚本语言：使用解释执行的编程语言

  • Python、JavaScript、PHP

静态语言： 编译器一次性生成目标代码，优化更充分，程序运行速度更快。
脚本语言：执行程序时需要源代码，维护更加灵活，跨平台操作。

6. 程序基本编写方法

IPO

I：Input输入

  • 文件输入、网络输入、控制台输入、交互界面输入、内部参数输入等

P：Process处理，程序的主要逻辑

  • 程序对输入数据进行计算产生输出结果的过程
  • 处理方法统称算法

O：Output输出

  • 控制台输出、图形输出、文件输出、网络输出操作系统内部变量输出等
  • 输出是展示运算结果的方式

1. 温度转换

温度刻画的两种不同体系

  • 摄氏度：中国等世界大多数国家使用
    以1标准大气压下水的结冰点为0度，沸点为100度，将温度进行等分刻画
  • 华氏度：美国、英国等国家使用
    以1标准大气压下水的结冰点为32度，沸点为212度，将温度进行等分刻画

  • 输入：带华氏或摄氏标志的温度值

  • 输出：带摄氏或华氏标志的温度值
  • 标识放在温度最后，F表示华氏度，C表示摄氏度，82F表示华氏82度，28C表示摄氏28度
  • 根据华氏和摄氏温度定义，利用转换公式如下：
      C = ( F – 32 ) / 1.8
  F = C * 1.8 + 32
    其中， C表示摄氏温度， F表示华氏温度

基本结构

前置物镜：成像目标成像于其焦平面内，收集成像目标的光信息。

  • 有效焦距：决定成像目标所占整个拍摄画面的比例大小

  • 视场角：决定前置物镜的视野范围大小

  • 相对孔径：决定前置物镜的纳光

  • 焦距较长的前置物镜的视场角较小，焦距较短的前置物镜的视场角则较大。长焦距的前置物镜适合拍摄较远距离的成像目标，短焦距的前置物镜适合拍摄距离较近的成像目标。

色散器件：使不同波长的光线以不同的规律入射到准直系统，省去扫描或者凝视所需的时间，实现快照式光谱成像

  • 透明基底表面附着了一层大小结构各不相同、尺寸为微米-纳米的透明例子

  • 透明基底表面分布着大小形状不同、尺寸为微米-纳米的凹陷

  • 或者上述两者共存

探测系统：采集光信号的强度，并将其接收到的光信号强度转化为电信号强度

准直系统

成像原理

分光原理： 光线入射到这些透明粒子或凹陷时表面会发生散射效应或者干涉效应。

  • 根据粒子的大小可分为米氏散射和瑞利散射

  • 米氏散射：入射光波长与散射例子相近

  • 瑞利散射:散射粒子的直径远小于入射光的波长

random 模块

  • 描述

    random（）方法为随机返回一个实数，其范围在[0，1）内。

    print (secret) 打印得到的结果
    以上实例得到结果：0.2964213966154332

  • random用法

    random各种使用方法

      import random

    随机生成[0.1)的浮点数

      print("random():", random.random())

    随机生成1000-9999之间的整数

      print("randint(1000, 9999):", random.randint(1000, 9999))

    随机生成0-20之间的偶数

      print("randrange(0, 21, 2):", random.randrange(0, 21, 2))

    随机生成0-20之间的浮点数

      print("uniform(0, 20):", random.uniform(0, 20))

    从序列中随机选择一个元素

      list_string = ['a', 'b', 'c', 'd', 'e']

  print("choice(list):", random.choice(list_string))

  print("choice(string):", random.choice('abcd'))

    对列表元素随机排序

      list_number = [1, 2, 3, 4, 5]

  random.shuffle(list_number)

  print("shuffle(list):", list_number)

光谱仪

发表于 2020-09-14 更新于 2020-09-22 分类于 光谱仪

@TOC

学习笔记

今天开始学习关于光谱仪的相关知识，希望能够在研一的这一年有所收获与成长。

名词解释

光谱技术：

光谱：电磁辐射按照波长的有序排列。注：根据实验条件不同，各个辐射的波长都具有各自的特征强度。

  • 复色光经过色散系统（棱镜、光栅）分光后，被色散的单色光按照波长或者频率大小依次排列的图案，全称即为光学光谱。

高光谱成像技术：高光谱成像技术是基于非常多窄波段的影像数据技术，它将成像技术与光谱技术相结合，探测目标的二维几何空间及一维光谱信息，获取高光谱分辨率的连续、窄波段的图像数据。

  • 高光谱图像：所谓高光谱图像就是在光谱维度上进行了细致的分割，不仅仅是传统所谓的黑、白或者R、G、B的区别，而是在光谱维度上也有N个通道，例如：我们可以把400nm-1000nm分为300个通道。因此，通过高光谱设备获取到的是一个数据立方，不仅有图像的信息，并且在光谱维度上进行展开，结果不仅可以获得图像上每个点的光谱数据，还可以获得任一个谱段的影像信息。

无序结构：当两种(或两种以上)原子或离子在晶体结构中占据某种位置时，如果它们相互间的分布是任意的,即它们占据任何一个该种位置的几率都是相同的，则这种结构称为无序结构

光谱分辨率：传感器中所能记录的电磁波谱中，某一特定的波长范围值，波长范围值越宽，光谱分辨率越低。

  • 例如，MSS多光谱扫描仪的波段数为5(指有5个通道)，波段宽度约为100～2 000 nm而成像光谱仪的波段数可达到几十基至几百个波段，波段宽度则为5～10 nm。
  • 波段宽度：光谱分辨率指成像的波段范围，分得愈细，波段愈多，光谱分辨率就愈高，技术可以达到0.17nm纳米）量级，400多个波段。细分光谱可以提高自动区分和识别目标性质和组成成分的能力。
  • 光谱分辨率：是指探测器在波长方向上的记录宽度，又称波段宽度(band width)。光谱分辨率被严格定义为仪器达到光谱响应最大值的50%时的波长宽度。
  • 注：传感器的波谱范围，一般来说识别某种波谱的范围窄，则相应光谱分辨率高。举个例子：可以分辨红外、红橙黄绿青蓝紫紫外的传感器的光谱分辨率就比只能分辨红绿蓝的传感器的光谱分辨率高。一般来说，传感器的波段数越多波段宽度越窄，地面物体的信息越容易区分和识别，针对性越强。

空间分辨率：遥感图像上能够识别两个相邻地物的最小距离。

  • 像元：将地面信息离散化而形成的网格单元，扫描影像的基本单元。
  • 像解率:单位距离能够分别的线宽或间隔相等的平行细线的长度。
  • 瞬时视场角：瞬时视场角（IFOV）是指传感器内单个探测元件的受光角度或观测视野，又称为传感器的角分辨率，单位为毫弧度(mrad)或微弧度(μrad)。瞬时视场角β与波长λ和收集器的孔径D有关：β=λ/2D

瞬时视场角越小，空间分辨率越高。

成像光谱仪

分类

按照波段数或光谱分辨率

  • 多光谱：波段数分布在几个到几十个谱段，光谱分辨率通常为几十到上百纳米。
  • 高光谱：光谱覆盖范围窄于多光谱性，波段数可达到几十到几百个谱段，光谱分分辨率通常为几纳米到几十纳米。
  • 超广谱：光谱覆盖范围更窄与高光谱成像光谱仪，波段数超过三百个连续谱段，光谱分辨率小于1nm。

色散型（色散元件不同）

  • 缺点：入射光的能量过于分散
  • 光栅型：
  • 棱镜型
    滤光片型
  • 基于多通带滤波片型
  • 基于空间 可见滤波片型
  • 基于可调谐滤波片型
    干涉型
  • 空间调整
  • 时间调制
